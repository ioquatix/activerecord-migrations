## Installation

Add this line to your application's Gemfile:

```ruby
gem 'activerecord-migrations'
```

## Usage

Add this to `tasks/db.rake`

```ruby
require 'activerecord/migrations'
ActiveRecord::Migrations.root = File.expand_path("../db", __dir__)
```
